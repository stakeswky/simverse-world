# Skills World — 设计与实施计划总览

> 一座 2D 像素风的赛博城市，任何人都能将真实人物、虚构角色或理想人格炼化为 AI 居民，让他们在这座永不关闭的城市里"活着"。

---

## 项目概况

| 项目 | 说明 |
|------|------|
| **类型** | 2D 像素风开放世界 + AI 角色扮演 + 社交经济系统 |
| **技术栈** | FastAPI + SQLAlchemy (Python) / React 18 + TypeScript + Phaser 3 |
| **核心玩法** | 用户炼化 AI 居民 → 走进赛博城市与居民/玩家对话 → Soul Coin 代币经济驱动 |
| **当前阶段** | Phase 1 MVP 代码基本完成，Phase 2 升级设计完成、实施计划就绪 |

### Phase 2: 系统升级 — 设计完成，待实施

> 设计文档: `01-design/02-forge-upgrade-design.md`（13 章，1332 行）

Phase 2 对系统进行全面升级，核心变更：

#### 1. 炼化管线升级（P2-2）

将炼化从"5 步引导对话"升级为智能管线：

```
快速炼化（30-60秒）: 用户输入 → LLM 生成三层 → 评分 → 完成
深度蒸馏（2-5分钟）: 智能路由 → 6路网络调研 → 三重验证提取 → 三层构建 → 质量验证 → 双Agent精炼
```

- 智能路由：自动判断走快速还是深度（公众人物/知名角色走深度，虚构角色走快速）
- 6 路调研：通过 SearXNG 自动搜索人物的著作、访谈、表达风格、外部评价、决策、时间线
- 三重验证：每个心智模型必须通过跨域复现、生成力、排他性三项检验
- 双 Agent 精炼：优化者 + 创造者两个视角独立审阅，提出改进建议

**已验证**：用萧炎（《斗破苍穹》）做了端到端测试，SearXNG 搜索 60 条结果（32s），LLM 生成 2903 字符（28s），输出质量优秀。见 `03-mvp-validation/`。

#### 2. 角色统一（P2-4）

**核心思路：玩家 = 角色**。每个用户都拥有自己的三层人设。

- 登录后引导创建角色（从零创建 / 加载预设 / 导入 Skill / 跳过）
- 支持导入外部 Skill 文件，自动格式检测 + 转换
- 14 个预设虚拟人物（乔布斯、马斯克、芒格、费曼、萧炎等）

#### 3. 对话系统升级（P2-5）

从"玩家→NPC 单向"升级为"玩家↔NPC + 玩家↔玩家"：

| 场景 | 响应方式 |
|------|---------|
| 玩家 → NPC | LLM + NPC 人设（现有） |
| 玩家 → 玩家（手动模式） | WebSocket 推送，对方自己回复 |
| 玩家 → 玩家（自动模式） | LLM + 对方人设自动回复 |
| 对方离线 + 自动 | LLM 异步回复 |
| 对方离线 + 手动 | 消息排队，上线后推送 |

#### 4. 视觉分层（P2-4）

- **头像/立绘**：AI 生成（Gemini Vertex AI），用于对话界面和角色卡
- **地图精灵**：从 25 个基础模板智能匹配 + 调色，保持像素风统一
- **出生点**：首次登录中央广场随机偏移，后续登录恢复上次位置

**已验证**：Gemini 生成的 Q 版头像质量良好，但不适用于 32×32 像素精灵表。见 `03-mvp-validation/sprite-gen-test-xiaoyan.png`。

#### 5. LinuxDo OAuth2 登录（P2-3）

标准 Authorization Code Grant 流程，直接对接 `connect.linux.do`，支持 trust_level 门槛。

#### 6. 管理面板（P2-6）

6 大模块覆盖系统全部管理需求：

| 模块 | 核心功能 |
|------|---------|
| 仪表盘 | 实时指标 + 7 日趋势 + 服务健康监控 |
| 用户管理 | CRUD + 余额调整 + 封禁 + 管理员设置 |
| 居民管理 | 三层人设编辑 + 预设人物管理 + 批量操作 |
| 炼化监控 | 活跃会话追踪 + SearXNG 健康 + Token 统计 |
| 经济系统 | 交易流水 + 参数动态配置（无需重启） |
| 系统配置 | LLM/热度/评分/街区/OAuth/精灵 — 全部可运行时调整 |

#### 7. 用户设置面板（P2-7）

| 设置组 | 内容 |
|--------|------|
| 账户 | 改名/改密/第三方绑定解绑/注销 |
| 角色 | 精灵图/头像/三层人设编辑/重新炼化/导入 Skill |
| 互动 | 自动/手动回复模式 + 离线代答 + 通知 |
| 隐私 | 地图可见性 + 人设公开范围 |
| LLM | 自定义 API Key（管理员开启后可用） |
| 经济 | 低余额提醒阈值 |

#### 8. LLM 归属分离

LLM 调用分为两类：

| 归属 | 场景 | 特征 |
|------|------|------|
| **系统 LLM** | 路由判断、质量评分、街区分配、格式检测 | 轻量（<500 token/次），平台承担 |
| **用户 LLM** | 对话、炼化、头像生成 | 重量级，用户可自带 API Key |

管理员可通过管理面板开启"允许用户自定义 LLM"，开启后用户可在设置中填写自己的 API Key，对话和炼化使用自己的 Key，不扣 Soul Coin。
