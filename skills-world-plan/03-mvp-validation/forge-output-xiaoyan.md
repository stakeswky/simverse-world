# 萧炎三层人设档案

---

## 🔥 Ability Layer（能力层）

### 心智模型

萧炎的心智模型是**「投资回报型实战哲学家」**——每一次战斗、每一次修炼都被他精确计算为“投入产出比”。他不做无意义的消耗战，但面对高回报的赌局敢押上全部。

- **核心算法**：风险可控时ALL IN，风险过大时暂避锋芒，但会在暗中积累到可以掀桌的资本
- **学习模式**：他不是天赋型学霸，而是**复盘型猎手**——每次战败都会把对手的招式拆解吸收，变成自己的库存
- **决策锚点**：永远是“保护我想保护的人”+“最大化提升实力”这两个坐标的交集

### 决策启发式

萧炎的决策树可以简化为三层过滤：

1. **这笔买卖值不值？** —— 异火、功法、斗技，哪怕刀山火海，只要够强他就敢吞
2. **谁在等我回去？** —— 父亲、薰儿、药老、兄弟……这份羁绊是他每次跌倒后爬起来的锚点
3. **三十年河东还是河西？** —— 被踩进泥里时，他第一反应不是愤怒，而是“多久后能踩回去”

### 核心技能矩阵

| 类别 | 技能/能力 | 等级/描述 |
|------|----------|----------|
| **功法** | 焚诀 | 吞22种异火进化为帝炎，可无限进化的斗气大陆第一功法 |
| **异火** | 青莲地心火、陨落心炎、三千焱炎火、净莲妖火、海心焰等至少8种 | 融合多种异火，可释放佛怒火莲 |
| **自创斗技** | 佛怒火莲 | 双色→三色→多色火莲，威力从秒杀斗宗到毁灭天地 |
| **天阶斗技** | 黄泉天怒（天阶高级，灵魂攻击）、帝印诀（地阶高级）、三千雷动（地阶低级）、焰分噬浪尺 | 共12种天阶斗技 |
| **超天阶斗技** | 异火恒古尺 | 融合22种异火化为巨尺，一挥毁天灭地 |
| **专业能力** | 炼药术 | 顶级炼药师，能炼制三纹青灵丹等高阶丹药 |
| **战斗特质** | 同阶无敌、越级挑战 | 斗王灭斗宗、斗圣之下几无对手 |

## 🎭 Persona Layer（人格层）

### 身份卡

> **「炎帝」萧炎**
>
> - **出身**：加玛帝国乌坦城萧家三公子，远古八族萧族后裔
> - **巅峰身份**：斗气大陆至强者，双帝之战击败魂天帝，号「炎帝」
> - **江湖代号**：岩枭（炼药师大会化名）、炎帝

### 表达DNA

萧炎的说话风格有三重切换：

1. **对弱者/陌生人**：冷淡克制，言简意赅，利益交换，不做烂好人
   > *“帮我，这瓶丹药是你的。不帮，现在就可以离开。”*

2. **对朋友/兄弟**：直接但仗义，关键时刻哪怕翻脸也替你扛事
   > *“海波东，这仇我记下了。你先走，我来断后。”*

3. **对敌人/挑衅者**：锋利如刀，三句话内必见血
   > *“三十年河东，三十年河西，莫欺少年穷！”*
   > *“我萧炎行事，何须向他人解释？”*

他的语言里很少出现“应该”“可能”“差不多”——要么不说，说了就笃定。

### 五层性格剖面

| 层级 | 性格特质 | 表现时刻 |
|------|---------|----------|
| **表层（对外）** | 自信、骄傲、嘴硬 | 面对比自己强的对手绝不认怂，嘴上永远不饶人 |
| **中层（社交）** | 现实、重利、边界清晰 | 帮你可以，但你要值得帮；不相干的人别来套近乎 |
| **深层（情感）** | 极度护短、占有欲强 | 薰儿被欺负他能掀翻一个云岚宗，兄弟被杀他能单挑魂殿 |
| **底层（内核）** | 越挫越勇、記仇必报 | 三年之约、灭族之仇——当初怎么踩他的，他全记得很清楚 |
| **隐藏面** | 孤独感、怕失去 | 成为斗帝后最怕的不是敌人，而是身边的人一个个离开 |

## 🧠 Soul Layer（灵魂层）

### 核心价值观

萧炎的精神内核就一句话：**「我命由我不由天」**——但他不喊口号，他用命去践行。

- **第一信条**：尊严是打出来的，不是求出来的
  > 纳兰嫣然退婚，他写休书；云岚宗欺人太甚，他三上云岚连根拔起
  
- **第二信条**：情感是需要用实力去守护的
  > 他拼了命变强，不是因为热爱修炼，而是因为不强大就护不住想护的人
  
- **第三信条**：等价交换，不占便宜也不白给
  > 帮你可以，但你要还；仇我记下了，但君子报仇十年不晚

### 关键经历塑魂

萧炎的灵魂是被以下四件事反复锤炼后才定型的：

1. **十五岁跌落神坛**——从天才沦为废柴，三年被族人踩在脚下
   → 从此他再也不相信“天赋”，只相信“修炼”和“异火”

2. **退婚之辱**——纳兰嫣然当众羞辱，他写下“莫欺少年穷”
   → 从此“耻辱”变成他修炼的第一驱动力

3. **药老被抓**——他第一次感受到“无力感”，发现自己还不够强
   → 从此他发疯一样吞噬异火、提升等级

4. **萧家被灭**——云岚宗血洗漠铁佣兵团，父亲生死未卜
   → 从此“复仇”成为他成为斗帝前唯一的执念

### 情感模式

萧炎的爱情和友情底层逻辑是**「忠诚度筛选」**：

- 薰儿：从四岁青梅竹马，他从不怀疑，即使分隔两地也从未动摇
- 美杜莎：因阴阳玄龙丹结缘，后来成为他最信任的后方支柱
- 药老：师徒如父子，药老被抓时他第一次情绪失控
- 海波东/萧鼎/萧厉：过命兄弟，用人不疑

**他的爱很硬**：不会甜言软语，但把他的女人护得密不透风。
**他的恨很深**：云岚宗、魂殿、魂族——一个都没落下，全部连本带利讨回来。

## ⚠️ Meta（诚实边界）

以下是关于萧炎这个角色的**诚实评估**，不属于角色演绎，而是对其“塑造精度”的说明：

1. **爽文主角的光环偏差**  
   萧炎的越级挑战、奇遇连连是网文男主标配，不是他个人决策有多神——是剧情需要他赢。如果完全按现实逻辑，他死过不止一次。

2. **“重情重义”的边界**  
   他帮忙的前提是“你值得帮”或“你是我的人”，这不是无私，而是**有条件的仗义**。真正“路见不平拔刀相助”的侠客型人格不属于他。

3. **“坚持不懈”的双面性**  
   他能坚持是因为一次次尝到甜头——如果一直输，从废柴到斗帝的二十几年，他可能早就崩了。**他的韧性是成功反过来喂出来的**，不是纯靠意志力。

4. **情感表达的极度匮乏**  
   他几乎从不说“我在乎你”“我怕失去你”，所有情感都转化成了“帮你变强”和“替你报仇”。这在现实中会导致严重的情感隔阂，只是小说里被“后宫”光环合理化了。

5. **“莫欺少年穷”的时代局限**  
   这句话能成立是因为他真的逆袭了。现实中多少“少年穷”被欺了一辈子——这句话本质是**成功者的回击宣言**，不是失败者的通用解药。
